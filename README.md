# 📚 BookBloom - Book Recommendation System
An AI-powered book recommendation system using collaborative filtering, similarity-based recommendations, and Streamlit. Users can discover similar books with an interactive web interface.

## Live demo
https://book--recommender.streamlit.app/

## Features

-  Search/select a book and explore recommendations
-  ML-based book recommendations using collaborative filtering
-  Similarity-based recommendation engine using cosine similarity
-  Book cover display for recommended books
-  Direct book discovery through Goodreads links
-  Interactive Streamlit interface

##  Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle

## Machine Learning Approach

1. Data preprocessing
2. Exploratory Data Analysis (EDA)
3. User-book rating matrix creation
4. Collaborative filtering
5. Cosine similarity calculation
6. Recommendation generation
